## 실행방법

1. 레포지토리 복제 후 의존성 설치

```
$ https://github.com/loveytheb/hanghae99_SNS_next.git
$ npm install
```

2. 개발 서버 가동

```
npm run dev
```

3. 브라우저에서 실행

```
http://localhost:3000/
```
